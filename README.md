## About The Project

This library was created to cater to the design requirements of Kodebi web applications. It provides a basic set of components like Buttons, Inputs, Dropdowns, etc. which come with the default Kodebi styles, but are also customizable to a certain degree. Future versions will provide even more components with even more customization options.

## Usage

In order to use the components in your project just do the following:

```javascript
import { Button } from '@kodebi/libkodebi-ui';

function App() {
	return <Button variant="action" type="submit" label="Test" />;
}
```
